# 🧠 AgentSkills

A curated collection of **AI Agent Skills** — plug-and-play instruction modules
that extend the capabilities of any AI coding assistant.

Each skill is a self-contained folder with a `SKILL.md` file that gives
AI agents rich, structured instructions to execute specialized tasks.

## 🚀 How to Use a Skill

### For Antigravity IDE

Copy the skill folder into your workspace's `.agents/skills/` directory:

```bash
# Example: install super-agent skill
cp -r super-agent/ your-project/.agents/skills/super-agent/
npx skills add https://github.com/DhamarPutra/AgentSkills --skill super-agent
```

Your AI agent will automatically discover and load the skill.

### For Other AI Agents (Cursor, Copilot, Claude, etc.)

1. Open the skill's `SKILL.md` file.
2. Copy its contents as a **custom instruction** or **system prompt** for your agent.
3. The agent will follow the skill's instructions when triggered by the keywords defined in the skill.

## 🗂 Skill Structure

Every skill follows a standard structure:

```
skill-name/
├── SKILL.md          ← Main instruction file (required)
├── README.md         ← Public-facing documentation (optional)
├── references/       ← Additional context documents (optional)
└── resources/        ← Templates, assets, scripts (optional)
```

### `SKILL.md` Format

```yaml
---
name: skill-name
description: >
  Brief description of what this skill does.
  Trigger keywords are also listed here.
---

# Skill Title

[Detailed instructions for the AI agent]
```

## 🤝 Contributing

Contributions are very welcome! To add a new skill:

1. Fork this repo
2. Create a new folder: `your-skill-name/`
3. Add a `SKILL.md` with the standard frontmatter format
4. Add a `README.md` explaining what the skill does and how to use it
5. Open a Pull Request

Please follow the existing skill structure to ensure compatibility with all agent tools.
